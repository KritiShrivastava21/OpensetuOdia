ଡକ୍ୟୁମେଣ୍ଟେସନ୍

ଓପେନସେତୁ: କିପରି କାମ କରେ

ଓପେନସେତୁ DNS over HTTPS (DOH) ଦ୍ୱାରା ସଚାଳିତ, ଯାହା ଏକ ନେଟୱାର୍କ ପ୍ରୋଟୋକଲ୍ ଯାହା HTTPS ମାଧ୍ୟମରେ ସୁରକ୍ଷିତ ଭାବେ ଡୋମେନ୍ ନାମଗୁଡିକୁ ସମାଧାନ କରେ। DOH ଗୋପନୀୟତା ଏବଂ ସୁରକ୍ଷା ବୃଦ୍ଧି କରେ, ମଧ୍ୟସ୍ଥମାନେ DNS ପ୍ରଶ୍ନଗୁଡିକୁ ଜାସୁସି କିମ୍ବା ବଦଳାଇ ପାରିବାକୁ ପ୍ରତିରୋଧ କରି।

ଭାରତ କା ଇଣ୍ଟରନେଟ୍

ଭାରତ କା ଇଣ୍ଟରନେଟ୍ Handshake ଉପରେ ନିର୍ମିତ, ଯାହା ଏକ open-source, decentralized naming protocol ଯାହା ଇଣ୍ଟରନେଟ୍ ପାଇଁ ପର୍ୟାୟ root zone ଭାବେ କାମ କରେ। ଏହା ସଂସ୍କରଣ-ରୋଧୀ, ଛେଡ଼ାଖାନା-ମୁକ୍ତ ଡୋମେନ୍ ସମାଧାନକୁ ସମର୍ଥନ କରେ। ଆମର ଇଣ୍ଟରନେଟ୍ ର backbone।

ନିକଟଭବିଷ୍ୟତରେ, ଆମେ ENS (Ethereum Name Service) ଏବଂ IPFS (InterPlanetary File System) ଭଳି ପ୍ରୋଟୋକଲ୍ ମଧ୍ୟ ସମର୍ଥନ କରିବାକୁ ଯୋଜନା କରୁଛୁ, ଯାହା ଅଧିକ open ଏବଂ interoperable web କୁ ସକ୍ଷମ କରିବ।

ନିଜ DOH ନିର୍ମାଣ କରନ୍ତୁ

ଯେକୌଣସି ବ୍ୟକ୍ତି ଏକ DOH resolver ନିର୍ମାଣ କରି ଭାରତୀୟ ଇଣ୍ଟରନେଟ୍ ଅନୁଗମନ କରିବା ଏବଂ ବିସ୍ତାର କରିବାକୁ ସମର୍ଥ। ସେହିଥିରେ ସ୍କୁଲ୍, ବିଶ୍ୱବିଦ୍ୟାଳୟ, ଷ୍ଟାର୍ଟଅପ୍, ନନ୍-ପ୍ରଫିଟ୍ କିମ୍ବା ସରକାରୀ ଏଜେନ୍ସୀ ଥାଏ। ଓପେନସେତୁ ମାତ୍ର ଏକ ଉଦାହରଣ; ଆପଣ ନିଜ ଡୋହ୍ ଚଲାଇପାରିବେ।
